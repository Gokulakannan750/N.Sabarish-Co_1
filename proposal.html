03
Technology Stack

Modern, lightweight, zero lock-in

Stable, open web standards chosen for speed, longevity and full code ownership — no proprietary platform, no recurring licence fees.

  • Semantic HTML5

    Accessible, well-structured markup that search engines and screen readers understand.

  • Modern CSS3

    Custom properties (design tokens), Grid & Flexbox, fluid clamp() typography and a mobile-first responsive system.

  • Vanilla JavaScript

    No framework overhead — IntersectionObserver reveals, counters, sliders and mobile nav in one small file.

  • Google Fonts & JSON-LD

    Sora + Inter type pairing and Schema.org structured data for SEO-rich search results.

Acceptance Criteria

The project is considered complete when all of the following are verified:

  1. All five pages render correctly on Chrome, Edge, Safari and Firefox.
  2. Layout adapts cleanly across mobile, tablet and desktop breakpoints.
  3. Mobile menu, sticky header and back-to-top button work on touch devices.
  4. Project gallery filters and image sliders function correctly.
  5. Status badges (completed / ongoing) display on the right projects.
  6. Enquiry form validates input and shows a success confirmation.
  7. Click-to-call, click-to-email and the embedded map all work.
  8. Every page has unique title, meta description and Open Graph tags.
  9. Schema.org structured data validates without errors.
  10. Scroll-reveal animations and stat counters trigger correctly.
  11. Site is live on the custom domain with favicon and HTTPS.
  12. All source files handed over and 30-day support window begins.
