# Client logos

Drop client logo files here using the exact filenames below. The homepage
"Trusted by leading institutions" marquee will automatically show the logo
once the file exists; until then it falls back to the client's text name.

Recommended: transparent PNG or SVG, roughly 200–400px wide, logo on a
transparent (or white) background. They are auto-scaled to ~42px tall.
